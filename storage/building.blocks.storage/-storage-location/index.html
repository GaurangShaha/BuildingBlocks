storage/building.blocks.storage/StorageLocation

StorageLocation

sealed class StorageLocation

Represents different storage locations available on an Android device for file operations. This sealed class allows for locations to be simple singletons (like InternalAppCache) or to carry data, such as a custom subdirectory path.
