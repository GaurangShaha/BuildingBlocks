Storage

object Storage

A singleton object that provides access to the main functionalities of the storage module.

It is responsible for initializing the required components and providing instances of FileRepository for both standard and encrypted file operations.

Important: You must call initialize before using any other methods in this object. This is typically done once in your Application's onCreate method.
