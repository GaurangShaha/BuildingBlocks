KeyPurpose

enum KeyPurpose : Enum<KeyPurpose>

Defines the intended use for a cryptographic key, which determines the alias used to store and retrieve it from the Android KeyStore.

Using distinct purposes for different cryptographic operations (e.g., encrypting text vs. encrypting files) helps to isolate keys and prevent key reuse, which is a security best practice.
