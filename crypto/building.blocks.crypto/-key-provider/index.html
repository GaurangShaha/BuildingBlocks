KeyProvider

object KeyProvider

Manages cryptographic keys within the Android KeyStore.

This object is responsible for creating, storing, and retrieving SecretKey instances used for encryption and decryption. It ensures that keys are securely handled by leveraging the Android KeyStore system, which provides protection against unauthorized key extraction.
