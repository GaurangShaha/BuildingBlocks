data class ExtraShape(val xlargeShape: Shape = GenericShape { _, _ -> }, val capsuleShape: Shape = GenericShape { _, _ -> })

Represents a collection of extra shapes beyond the standard ones provided by Compose.

This data class holds references to custom shapes that can be used within Compose UI elements. It provides a convenient way to define and reuse custom shapes throughout your application.
